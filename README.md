<h2>🛠 Tech Stack</h2>
<p>このポートフォリオサイトは以下の技術を使用して開発されています。</p>
<ul>
  <li><strong>Framework:</strong> <a href="https://nextjs.org/" target="_blank">Next.js</a> (App Router)</li>
  <li><strong>Language:</strong> <a href="https://www.typescriptlang.org/" target="_blank">TypeScript</a></li>
  <li><strong>Package Manager:</strong> <a href="https://pnpm.io/" target="_blank">pnpm</a></li>
  <li><strong>Deployment:</strong> <a href="https://pages.cloudflare.com/" target="_blank">Cloudflare Pages</a></li>
</ul>

<h2>🚀 Getting Started</h2>
<p>ローカル環境でこのプロジェクトを実行するための手順です。</p>

<h3>1. リポジトリのクローン</h3>
<pre><code>git clone https://github.com/YutaNagashima1031/portfolio.git
cd portfolio</code></pre>

<h3>2. パッケージのインストール</h3>
<pre><code>pnpm install</code></pre>

<h3>3. 開発サーバーの起動</h3>
<pre><code>pnpm dev</code></pre>
<p>ブラウザで <a href="http://localhost:3000" target="_blank">http://localhost:3000</a> を開くと、結果を確認できます。</p>

<h2>☁️ Deployment on Cloudflare Pages</h2>
<p>このプロジェクトは <strong>Cloudflare Pages</strong> にデプロイされています。<br>
メインブランチ（<code>main</code>）に変更をプッシュすると、自動的にビルドとデプロイが実行されます。</p>
